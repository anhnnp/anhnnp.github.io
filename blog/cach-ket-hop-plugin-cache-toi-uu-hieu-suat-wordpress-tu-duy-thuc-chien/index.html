Trong quá trình tối ưu tốc độ cho WordPress, mình nhận ra đa số anh em chỉ bật plugin theo bản năng: bật hết cache, bật hết minify, bật hết lazyload → hy vọng web chạy nhanh. Nhưng cuối cùng:

  • web chậm hơn
  • layout vỡ
  • WooCommerce lỗi giỏ hàng
  • Core Web Vitals tụt điểm

Vấn đề không nằm ở plugin. Vấn đề nằm ở kết hợp sai. Nên kết hợp chúng trong trường hợp nào, không nên kết hợp khi nào và cách dùng thế nào để đạt hiệu suất tối ưu.

Trong bài này, mình sẽ chia sẻ góc nhìn thực tế về 7 plugin nổi tiếng:

  • FlyingPress
  • WP Rocket
  • LiteSpeed Cache
  • Super Page Cache (cho Cloudflare)
  • Perfmatters
  • WP-Optimize
  • W3 Total Cache

I. Phân loại plugin theo chức năng

1. Plugin Cache Fullstack

✔ FlyingPress ✔ WP Rocket ✔ LiteSpeed Cache ✔ W3 Total Cache

Những plugin này phụ trách:

  • Page cache
  • Minify
  • Defer/Delay JS
  • Lazyload
  • Remove unused CSS
  • Font optimization
  • CDN (tùy plugin)

NGUYÊN TẮC SỐ 1: Chỉ chạy 1 plugin cache fullstack.

Chạy 2 plugin cùng loại = xung đột chắc chắn.

2. Plugin Cache ở tầng Cloudflare

✔ Super Page Cache

→ Plugin này đẩy cache HTML lên Cloudflare Edge. → Không minify, không delay JS → không trùng với plugin tối ưu khác.

Có thể kết hợp với Perfmatters / WP-Optimize.

3. Plugin tối ưu asset & script

✔ Perfmatters

Plugin này không cache HTML, mà tập trung vào:

  • Remove unused CSS
  • Delay JS
  • Disable scripts per page
  • Tắt embed, emoji, heartbeat
  • Preload / preconnect
  • Giảm bloat từ theme & plugin

Perfmatters kết hợp tốt với hầu hết plugin cache.

4. Plugin dọn dẹp database, media, table

✔ WP-Optimize

Chuyên làm:

  • Clean DB
  • Remove revision
  • Compress image
  • Optimize table
  • Tự động cleanup

Không tối ưu JS/CSS → không gây xung đột.

II. Phân tích từng plugin & cách kết hợp

1. FlyingPress

Điểm mạnh:

  • Remove unused CSS tốt nhất hiện nay
  • Delay JS ổn định
  • Tối ưu ảnh, font, iframe mạnh
  • Cấu hình rất đơn giản
  • Ít gây lỗi theme/plugin

Không nên kết hợp với: ❌ WP Rocket ❌ LiteSpeed Cache ❌ W3 Total Cache ❌ FlyingPress + Super Page Cache (vì 2 tầng cache HTML → xung đột)

Nên kết hợp với: ✔ Perfmatters (tắt Lazyload trong Perfmatters) ✔ WP-Optimize (dọn DB hằng tuần) ✔ Cloudflare (nhưng tắt Auto-minify, Rocket Loader)

Phù hợp cho:

  • WooCommerce
  • Website quốc tế
  • Blog + landing page
  • VPS Nginx / Apache

2. WP Rocket

Điểm mạnh:

  • Dễ dùng
  • Cache ổn định
  • Preload sitemap
  • Lazyload, delay JS khá tốt
  • Không gây lỗi vặt

Không nên kết hợp với: ❌ LiteSpeed Cache ❌ FlyingPress ❌ W3 Total Cache ❌ Super Page Cache (trùng cache HTML)

Nên kết hợp với: ✔ Perfmatters (tắt minify ở 1 trong 2) ✔ WP-Optimize ✔ Cloudflare (tắt Auto-minify)

Phù hợp cho:

  • Newbie
  • Blog, tin tức
  • Web nhỏ – trung bình
  • Hosting shared

3. LiteSpeed Cache

Điểm mạnh:

  • Cache ở mức server → rất nhanh
  • QUIC.cloud mạnh
  • Image optimization free
  • Lazyload, critical CSS, cache vary rất tốt

Không nên kết hợp với: ❌ WP Rocket ❌ FlyingPress ❌ W3 Total Cache ❌ Super Page Cache (cache vs cache)

Nên kết hợp với: ✔ Perfmatters (chỉ dùng Script Manager + delay JS) ✔ WP-Optimize

Phù hợp cho:

  • Server LiteSpeed
  • Hostinger / A2 Hosting / HawkHost
  • WooCommerce lớn
  • Web traffic cao

Nếu không dùng LiteSpeed server → KHÔNG dùng LSCache.

4. Super Page Cache (Cloudflare)

Điểm mạnh:

  • Cache HTML lên Cloudflare Edge → tốc độ tốt ở mọi quốc gia
  • Free
  • Không tối ưu JS/CSS → dễ kết hợp

Không nên kết hợp với: ❌ FlyingPress (trùng cache HTML) ❌ WP Rocket ❌ LiteSpeed Cache ❌ W3 Total Cache

Nên kết hợp với: ✔ Perfmatters → tối ưu JS/CSS ✔ WP-Optimize → dọn DB ✔ Cloudflare APO (hoặc không cần)

Phù hợp cho:

  • Website quốc tế
  • VPS Nginx / Apache
  • Người muốn tận dụng Cloudflare miễn phí

5. Perfmatters

Điểm mạnh:

  • Tối ưu JS/CSS
  • Delay JS mạnh nhất
  • Script Manager = tắt bloat theo page
  • Tắt các tính năng WP vô ích
  • Tinh chỉnh preload, preconnect

Không nên kết hợp với: ❌ Minify trùng lặp từ nhiều plugin (Chỉ để Perfmatters hoặc plugin cache xử lý)

Nên kết hợp với hầu hết plugin cache: ✔ FlyingPress ✔ WP Rocket ✔ LiteSpeed Cache ✔ Super Page Cache ✔ WP-Optimize

Phù hợp cho:

  • WooCommerce
  • Landing page
  • Blog nặng JS
  • Theme page builder (Elementor, Bricks, Divi)

6. WP-Optimize

Điểm mạnh:

  • Cleanup database
  • Compress image
  • Optimize table
  • Auto cleanup

Không gây xung đột với: ✔ FlyingPress ✔ WP Rocket ✔ LiteSpeed Cache ✔ Super Page Cache ✔ Perfmatters ✔ W3TC

Không nên bật: ❌ Minify ❌ Lazyload ❌ Cache (tính năng yếu)

Phù hợp cho:

  • Mọi website
  • Web lâu năm có nhiều revision
  • WooCommerce store

7. W3 Total Cache

Điểm mạnh:

  • Tối ưu sâu
  • Hỗ trợ Redis, Memcached
  • Tùy chỉnh rất nhiều layer

Không nên kết hợp với: ❌ WP Rocket ❌ FlyingPress ❌ litespeed Cache ❌ Super Page Cache ❌ Perfmatters delay JS trùng lặp (cần tinh chỉnh cẩn thận)

Nên kết hợp với: ✔ WP-Optimize ✔ CDN Cloudflare (không bật minify)

Phù hợp cho:

  • Dev
  • Website doanh nghiệp
  • Server custom (Redis, Memcache)
  • Ai cần kiểm soát chi tiết từng layer cache

IV. Gợi ý combo tối ưu cho từng loại website

1. WooCommerce lớn

➡ FlyingPress + Perfmatters + WP-Optimize hoặc ➡ LiteSpeed Cache + Perfmatters + WP-Optimize (nếu server LS)

2. Website quốc tế (US, UK, CA, AU traffic)

➡ Super Page Cache + Perfmatters + WP-Optimize

3. Blog cá nhân, tin tức

➡ WP Rocket + Perfmatters + WP-Optimize

4. Landing page / POD store

➡ FlyingPress + Perfmatters

5. Hosting shared

➡ WP Rocket + WPO hoặc ➡ LiteSpeed Cache + WPO (nếu LS server)

6. Website cần tối ưu sâu (dev)

➡ W3 Total Cache + WP-Optimize + Perfmatters (tắt delay trùng)

V. Tổng kết

  • Plugin cache → chỉ dùng 1 cái.
  • Plugin tối ưu asset → dùng Perfmatters là lựa chọn tối ưu nhất.
  • Plugin dọn database → WP-Optimize là lựa chọn ổn định và ít lỗi.
  • Cloudflare cache → dùng Super Page Cache nếu bạn cần tốc độ đa quốc gia.
  • Dùng FlyingPress hoặc LiteSpeed Cache tuỳ vào server.
  • W3 Total Cache chỉ dành cho dev hoặc hệ thống custom.

Công thức chìa khóa: Cache = 1 plugin duy nhất Optimization = Perfmatters Clean DB = WP-Optimize
