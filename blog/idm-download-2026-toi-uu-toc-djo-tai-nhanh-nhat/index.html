Bài viết được biên soạn lại bởi Phan Anh

Mình hay tải rất nhiều file: tài liệu, video, bộ cài, archive lớn. Trước đây IDM là lựa chọn “một cú nhấp” để tăng tốc tải nhưng nếu bạn muốn giữ mọi thứ hợp pháp hoặc không muốn tốn tiền, vẫn có nhiều thủ thuật và công cụ miễn phí rất mạnh. Bài này mình sẽ hướng dẫn bạn: (1) cài IDM chính thức + cách dùng trial dài hạn; (2) các lựa chọn thay thế miễn phí & cách cấu hình để tải nhanh; (3) mẹo tối ưu mạng và trình duyệt để giảm lỗi và tăng tốc.

Cài IDM chính thức & cách dùng miễn phí dài hạn

  • Tải phiên bản chính thức từ trang chủ Internet Download Manager. IDM có trial 30 ngày để bạn đánh giá trước khi mua.
  • Cài đặt IDM từ nguồn chính thức.
  • Mở IDM và làm quen với giao diện: thêm link tải, chọn thư mục đích, cấu hình nối tiếp kết nối, v.v.
  • Sử dụng các chức năng mở rộng trong IDM (bắt link từ trình duyệt, tích hợp trình duyệt, quản lý queue).

IDM Activation Script mở khoá tính năng

IDM Activation Script là một công cụ giả lập phiên bản license của Internet Download Manager (IDM). Lưu ý: Chúng tôi không bao gồm hoặc khuyến khích bất kỳ hành vi vi phạm bản quyền nào. Luôn sử dụng phần mềm theo giấy phép chính thức của nhà phát triển.

Phiên bản

Coporton IDM Activation Script 2.5.7 (Nguồn chính thức Coporton Github) (Hỗ trợ tham khảo cho Internet Download Manager 6.42 Build 52)

  • Giao diện đơn giản, dễ sử dụng.
  • Có kiểm tra tương thích giữa phiên bản công cụ và phiên bản IDM (chỉ thông báo tương thích).
  • Hỗ trợ nhiều phiên bản IDM (mô tả tương thích).
  • Nhẹ và chạy nhanh.

Hướng dẫn tải IDM Activation Script

Cách 1: (Khuyến khích)

  1. Mở PowerShell:

    • Click chuột phải vào Start menu và chọn Windows PowerShell or Windows Terminal.
    • Nếu có xác nhận User Account Control (UAC) thì hãy chọn Yes để cho phép PowerShell chạy với quyền Administrator cao nhất.

  2. Copy và Paste vào PowerShell dòng lệnh như dưới đây sẽ tải script và chạy tự động tất cả cho đến lúc hiển thị tool sử dụng.

irm https://coporton.com/ias | iex

Cách 2:

  1. Tải thủ công: Vào github và vào release source code để tải về.
  2. Giải nén: Giải nén file đã tải về.

Sử dụng tool

1. Chạy tool script:

  • Hãy click vào IASL.cmd để chạy nó. Tool này sẽ yêu cầu quyền chạy như Administrator nếu chưa được thiết lập vì vậy nếu hỏi thì bạn cần chọn “Run as Administrator.”
  • Nếu có cần chọn User Account Control (UAC), hãy chọn Yes để gán đầy đủ quyền chạy cho tool.
  • Một khi đã đầy đủ quyền, tool sẽ hiển thị trong màn hình hiển thị như bước (2).

2. Màn hình hướng dẫn trong tool sẽ hiển thị:

  • Các lựa chọn trong menu hiển thị trong tool:
    • 1 to Download Latest IDM Version.
    • 2 to Activate Internet Download Manager.
    • 3 to Extra FileTypes Extensions.
    • 4 to Do Everything (2 + 3).
    • 5 Clean Previous IDM Registry Entries.
    • 6 to exit.
  • Nếu chọn 4, tool sẽ copy các file cần thiết vào vị trí cài đặt của IDM và sẽ báo cáo tiến độ hoàn thành cũng như kết quả giả lập license IDM khi hoàn thành.

Hoàn thành!

Nguồn bài viết từ https://github.com/Coporton/IDM-Activation-Script
